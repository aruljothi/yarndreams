# yarndreams
Yarn Dreams by Aruna

## Published site (GitHub Pages)

The site can be published through GitHub Pages. After enabling Pages for the `main` branch (root), your site will be available at:

https://aruljothi.github.io/yarndreams/

If you prefer the site to open on a specific page, the repository includes an `index.html` that redirects to `crochet.html`.

## Quick publish / update instructions

1. Make any edits locally, then commit and push to `main`:

```bash
git add .
git commit -m "Publish: update crochet page and index redirect"
git push origin main
```

2. In GitHub: go to Settings â†’ Pages, set source to `main` branch and the folder to `/ (root)` and Save.

3. Wait a minute, then visit: `https://aruljothi.github.io/yarndreams/`.

4. If you want automatic builds and previews, you can connect this repo to Netlify or Vercel as an alternative.
